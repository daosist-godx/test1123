# Поиск запрещенных файлов у пользователей

Набор для удаленного запуска `Everything` на ПК пользователя и последующего поиска файлов по имени через ETP-подключение.

## Что делает

- копирует `EverythingPortable` на удаленный ПК в `C:\!install\Tools\Everything`;
- запускает `Everything.exe` на удаленном ПК через `PsExec`;
- после этого ты подключаешься к удаленному ETP-серверу из локального `Everything64.exe`;
- дальше ищешь файлы в интерфейсе `Everything` как обычно.

## Как работает `PsExec`

- `robocopy` копирует portable-папку на удаленный ПК по `\\HOST\c$`;
- `PsExec.exe` запускает на удаленном ПК `Everything.exe`;
- ключи в BAT:
- `-u` и `-p` — учетные данные администратора;
- `-s` — запуск от `SYSTEM`;
- `-d` — не ждать завершения процесса;
- `-accepteula` — принять EULA без отдельного окна.

## Что нужно настроить

В `Remote Files Search (запуск от имени админа).bat`:
- `src` — путь к локальной папке `EverythingPortable`;
- `userName` — логин администратора по умолчанию;
- `remoteInstallDir` — путь установки на удаленном ПК.

В `EverythingPortable/Everything.ini`:
- `etp_server_port=14630` — порт ETP;
- `etp_server_username=CHANGE_ME` — логин для ETP;
- `etp_server_password=CHANGE_ME` — пароль для ETP.

Если меняешь путь установки, поправь и:
- путь назначения в строке `robocopy`;
- путь к `Everything.exe` в строке с `PsExec`.

## Креды по умолчанию

- в опубликованной версии реальные креды удалены;
- `userName` в BAT — шаблон `DOMAIN\ADMIN_USER`;
- пароль администратора вводится вручную при запуске;
- ETP-логин и пароль в `Everything.ini` — `CHANGE_ME` и `CHANGE_ME`.

## Условия для работы

- нужен доступ к удаленному ПК по сети;
- нужен доступ к `\\HOST\c$`;
- нужны права локального администратора на удаленном ПК;
- для `PsExec` обычно нужен `TCP 445` и рабочие admin shares;
- для подключения клиента `Everything` нужен открытый `TCP 14630`;
- firewall на удаленном ПК не должен блокировать SMB и ETP;
- `Everything.exe` должен успешно стартовать на удаленном ПК.

## Как запускать

1. Заполни свои значения в BAT и `Everything.ini`.
2. Запусти `Remote Files Search (запуск от имени админа).bat` от администратора.
3. Введи IP или имя ПК.
4. Введи логин администратора или просто нажми Enter для значения из `userName`.
5. Введи пароль администратора.
6. После успешного запуска открой `EverythingPortable\Everything64.exe`.
7. В `Everything`: `Сервис -> Подключиться к ETP-серверу`.
8. Укажи:
- хост: IP или имя удаленного ПК;
- порт: `14630`;
- логин и пароль: из `Everything.ini`.

## Что входит в папку

- `Remote Files Search (запуск от имени админа).bat` — запуск копирования и удаленного старта;
- `PsExec.exe` — удаленный запуск процесса;
- `EverythingPortable/` — клиент и серверная portable-сборка `Everything`.
