# Название скрипта

## Что делает

Коротко в 2–4 пунктах:
- что запускает;
- с чем работает;
- какой результат дает.

## Как запускать

```bat
или
```

```powershell
команда запуска
```

Пошагово:
1. что ввести;
2. что выбрать;
3. какой ожидается результат.

## Особенности

- на какой ОС работает;
- нужны ли права администратора;
- какие встроенные утилиты или внешние зависимости используются.

## Узкие места

- что чаще всего ломается;
- какие порты, права, службы или модули нужны;
- что пользователь должен проверить в первую очередь.

## Что можно улучшить потом

- проверки ошибок;
- параметры командной строки;
- логирование;
- более безопасную работу с учетными данными.
