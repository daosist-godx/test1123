# RDS shadow (запуск от админа).bat

## Что делает

Минимальная BAT-обертка над `mstsc /shadow`.

Скрипт:
- спрашивает IP удаленного ПК;
- показывает доступные сессии через `query session`;
- просит ID нужной сессии;
- запускает shadow-подключение с управлением.

## Как запускать

```bat
RDS shadow (запуск от админа).bat
```

Дальше:
1. введи IP удаленного ПК;
2. посмотри список сессий;
3. введи `RDP user ID`;
4. дождись запуска окна подключения.

## Особенности

- использует `query session /server:<IP>`;
- использует `mstsc /shadow:<ID> /v:<IP> /control`;
- сам по себе не меняет политики и реестр.

## Узкие места

- целевой ПК должен быть доступен по сети;
- обычно нужен доступ к RDP-службам на целевом хосте;
- у администратора должны быть права на shadow-подключение;
- нужная сессия должна уже существовать;
- на некоторых хостах потребуется подтверждение пользователя.

## Что можно улучшить потом

- добавить проверку пустого IP и пустого ID;
- добавить понятное сообщение при ошибке `query session`;
- сделать отдельную версию с `/noConsentPrompt` для сред, где это заранее разрешено политикой.
