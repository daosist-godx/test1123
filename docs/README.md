# Документация по тулзам

Здесь лежат короткие README по скриптам и наборам из репозитория.

Идея простая: сами скрипты остаются в твоем стиле, а рядом есть нормальная шпаргалка — что делает тулза, как запускать и где у нее тонкие места.

## Что уже описано

- `network_scan.ps1` — `docs/network_scan.md`
- `MSG (запуск от админа).bat` — `docs/msg.md`
- `RDS shadow (запуск от админа).bat` — `docs/rds-shadow.md`
- `Поиск запрещенных файлов у пользователей/` — `docs/remote-files-search.md`
- `PSEXEC (запуск от админа).bat` — `psexec/README.md`

## Как это использовать дальше

Для новых скриптов логика такая:
- сам файл кидаешь как тебе удобно;
- рядом или в `docs/` добавляешь короткий README;
- пишешь без воды: что делает, как запускать, особенности, узкие места.

Так репа остается живой и не превращается в склад непонятных батников.
