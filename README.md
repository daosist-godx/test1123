# Scripts

Набор небольших утилит. Сейчас в репозитории:

## `network_scan.ps1`

Сканирует подсеть `/24`, пингует хосты, пытается получить имя компьютера, текущего пользователя и `DisplayName` из AD. Результат сохраняет в `TXT` или `CSV`.

### Главное

- диапазон сканирования: `1..254`;
- формат вывода: `TXT` или `CSV`;
- поля результата: `IP`, `Hostname`, `User`, `DisplayName`.

### Запуск

```powershell
powershell -ExecutionPolicy Bypass -File .\network_scan.ps1
```

При запуске введи подсеть в виде `192.168.1` или `10.0.0`, затем выбери формат `1` или `2`.

### Если другая подсеть

- для другой `/24` код менять не нужно: просто введи другой префикс при запуске;
- если хочешь зашить подсеть в код, измени строку `$subnet = Read-Host ...` на нужное значение;
- если нужен другой диапазон адресов, измени строку `1..254 | ForEach-Object {`.
